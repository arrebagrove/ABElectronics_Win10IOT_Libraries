Demo GUI Application

The "DemoApplication" directory contains a sample GUI application to connect to each of the supported boards and get and set data.

The application has a separate screen for each expansion board and class with buttons and slider controls to allow you to change and read/set values for the selected expansion board.

The project/solution is written for Visual Studio 2015 running under Windows 10.

The "ABElectronics_Win10IOT_Libraries.dll" dll will need to be built and included in your projects to use with the demo and test applications.
